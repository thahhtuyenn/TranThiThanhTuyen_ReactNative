
# 📝Bài tập tuần 6
### Nội dung
Code giao diện login với username nếu username không rỗng thì chuyển sang trang hiển thị danh sách các công việc chưa hòa tất, thêm công việc, sửa công việc và thay đổi trang thái đối với công việc đã hoàn thành.
### Hướng dẫn
- Sau khi clone về, mở VisualCode và mở thư mục ReactNativeLab06 
- Mở terminal và thực thi các lệnh sau: 
install tất cả các dependence có trong file package.json
```bash
    npm i 
```
Start project lab 6 sau khi mở thư mục ReactNativeLab06 với lệnh 
``` bash
    npm start
```
Để chạy trên nền tản web sau khi start project thành công thì nhấn phím 'w' còn nếu chạy trên AndroiStudio có thể nhấn 'a'

- Sau khi start sẽ hiển thị giao diện login, người dùng nhập username, nếu user không bị rỗng sẽ chuyển sang trang chủ, hiển thị danh sách todo, người dùng chọn nút (+) bên dưới để thêm 1 todo mới, chọn icon sửa ở ngay cạnh phải công việc để chỉnh sửa title của công việc, nếu việc hoàn thành người dùng có thể chọn check ở bên trái công việc để thay đổi trạng thái công việc thành đã hoàn thành.
